# AppleDevice

## Usage
### Installation with CocoaPods
```ruby
pod 'AppleDevice'
```

### Example
```objc
NSLog(@"%@", [AppleDevice currentDevice].deviceModel);//iPhone11,2
NSLog(@"%@", [AppleDevice currentDevice].deviceModelName);//iPhone XS
```
